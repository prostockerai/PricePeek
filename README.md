\## 📋 Overview



SmartShop BD is a professional price comparison web application for Bangladesh that automatically scrapes and compares product prices from major Bangladeshi online marketplaces.



\### Supported Marketplaces

\- 🛍️ \*\*Daraz\*\* - Bangladesh's largest online marketplace

\- 💻 \*\*StarTech\*\* - Leading computer \& electronics retailer

\- 🖥️ \*\*Ryans\*\* - Popular computer \& accessories store

\- 📱 \*\*Pickaboo\*\* - Electronics \& gadgets

\- 🔧 \*\*Gadget \& Gear\*\* - Tech accessories

\- 🏪 \*\*TechLand\*\* - Computer hardware

\- 🛒 \*\*Othoba\*\* - General marketplace

\- 📦 \*\*AjkerDeal\*\* - Deals \& discounts



\## ✨ Features



\### Core Features

\- 🔍 \*\*Live Product Search\*\* - Real-time scraping from multiple marketplaces

\- 💰 \*\*Price Comparison\*\* - Compare prices side-by-side

\- 🏷️ \*\*Best Deal Detection\*\* - Automatic best deal highlighting

\- 🎫 \*\*Coupon Finder\*\* - Available coupon codes with one-click copy

\- 💳 \*\*Cashback Offers\*\* - bKash, Nagad, Visa, Mastercard offers

\- ❤️ \*\*Wishlist\*\* - Save products locally (no account needed)

\- ⚖️ \*\*Product Comparison\*\* - Compare up to 5 products

\- 📱 \*\*Mobile Responsive\*\* - Works on all devices

\- 🔄 \*\*Auto Refresh\*\* - Prices refresh every 15 minutes



\### Technical Features

\- ⚡ \*\*Caching System\*\* - Reduces requests and improves speed

\- 🛡️ \*\*Rate Limiting\*\* - Prevents blocking

\- 🔄 \*\*Proxy Rotation\*\* - Multiple user agents

\- 📊 \*\*Error Handling\*\* - Graceful fallback when scraping fails

\- 🎨 \*\*Modern UI\*\* - Professional SaaS-style interface
